3. 配件详情：
* 主控：ESP32-S3
* 音频：MAX98357
* 电池管理：IP5306
* 3.3V稳压：Asm1117
* 氛围灯：WS2812B
* 温湿度传感器：AHT20
* 陀螺仪：MPU6050
* 屏幕：ST7789 1.3寸IPS
